Manuál

Předpokládejme, že máte operační systém Linux, máte nainstalovaný git, jste superuživatel root nebo máte právo používat příkaz sudo.

Edithosts nemůžete v žádném případě používat na systému Windows (ne, opravdu ani na Windows XP, Windows Vista, Windows 7, Window 8, Windows 9, Windows 10, Windows 11, Windows 12, ani na jakékoliv superbudoucí verzi Woken, snad jen kdyby společnost Microsoft začala své nové produkty dělat na linuxovém základě).

Instalace:

cd /tmp
git clone https://github.com/jiri001meitner/edithosts.git
edithosts/install.sh
Slouží k instalaci programu a zálohování aktuálního nastavení v /etc/hosts.

Odinstalace:

/opt/edithosts/uninstall.sh
Slouží k odinstalaci programu a vrácení všech nastavení.

S instalací a odinstalací si většina uživatelů vystačí.

Pro pokročilejší

/opt/edithosts/temporarilydisableblocking.sh
nebo 
temporarilydisableblocking
Slouží pro dočasnou deaktivaci blokování, ta je nastavena na 10 minut (pokud vám nastavená doba nevyhovuje, můžete jí změnit v souboru /opt/edithosts/temporarilydisableblocking.sh. Když vám nejede nějaký web, tak takhle můžete ověřit, zdali není na černé listině. Pokud zmáčknete v terminálu Ctrl+C (^C), tak Edithosts zůstane zablokovaný tak dlouho, dokud ho ručně neprobudíte přes příkaz refreshblocking. 

/opt/edithosts/refreshblocking.sh
nebo 
refreshblocking
Slouží pro ruční aktivaci blokování, pokud jste deaktivovali prostřednictvím temporarilydisableblocking.

Ruční úprava souboru /etc/hosts

/opt/edithosts/edithosts.sh
Slouží pro editaci uživatelské části /etc/hosts (části bez černé listiny, ta je oddělená) ve vašem oblíbeném editoru. Přímá editace souboru /etc/hosts se nedoporučuje, protože při aktualizaci by došlo k vymazání změn.

Další příkazy, info o souborech

/etc/cron.monthly/updateprogramuedithosts
Je symbolickým odkazem k /opt/edithosts/autoupdate.sh a slouží k automatické měsíční aktualizaci programu. 

/etc/cron.weekly/edithosts
Je symbolickým odkazem k /opt/edithosts/cron.weekly-edithosts a slouží k týdenní automatické aktualizaci černé listiny. 

/opt/edithosts/edithosts.sh
nebo 
edithosts
Slouží pro editaci /etc/hosts ve vašem oblíbeném editoru. 

/opt/edithosts/gethosts.sh
Slouží pro ruční aktualizaci černé listiny a sloučení s uživatelsky editovatelnou částí, tento script je spouštěn ostatními částmi prorgamu, ručně ho není potřeba spouštět.
